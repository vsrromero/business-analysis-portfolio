# Business Analysis Case Studies

This repository contains anonymized examples of business analysis documentation created during my recent professional work as a Data Analyst / Business Analyst.

The documents include feature specifications, Gherkin scenarios, improvement reports, and bug analysis produced during requirements analysis, product discussions, and process improvement initiatives.

All company names, systems, and internal identifiers were replaced with generic placeholders (FOO, BAR, ACME, etc.) to preserve confidentiality.

## Case Studies

1. [Attendance CSV Export Feature](./ba-documents/features/Attendance_CSV_Export_SPEC.md)  
   Definition of CSV export functionality for workforce attendance data, including acceptance criteria and data structure.

2. [AI Workforce Reporting Feature](./ba-documents/features/AI_Workforce_Reporting_SPEC.md)  
   Feature specification for an AI-powered reporting assistant capable of generating workforce reports based on natural language prompts.

3. [Weekly Attendance Calendar Bug Analysis](./ba-documents/bug-reports/Weekly_Attendance_Calnedar_Display_BUG.md)  
   Investigation of a calendar display issue affecting weekly attendance views, including expected behaviour and technical recommendations.

4. [Employee List Navigation Improvement ](/ba-documents/features/Attendance_List_Navigation_And_Pagination_SPEC.md)
   Usability and navigation improvements for the attendance employee list, focusing on pagination, table navigation, and monitor optimization.
